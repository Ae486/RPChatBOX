## 🔧 使用方式

### 查看性能统计（调试）

性能数据会**自动输出到控制台**，无需手动操作：
```
[渲染性能] 渲染23次 | 缓存命中率85.2% | 平均12.5ms | 提升8.5x | 缓存15/100项
```

### 手动打印性能报告

```dart
import 'package:chatbox/rendering/utils/performance_monitor.dart';

PerformanceMonitor().printReport();
```

输出：
```
=== 渲染性能报告 ===
总渲染次数: 50
平均渲染时间: 12.5 ms
缓存命中率: 85.20%
缓存命中平均时间: 2.3 ms
缓存未命中平均时间: 52.1 ms
性能提升: 22.65x
缓存使用: 23 项
====================
```

### 清除缓存

```dart
import 'package:chatbox/rendering/core/render_cache.dart';

RenderCache().clear();
```

## 📈 数据分析

### 缓存效果预测

假设用户查看包含100条消息的对话：
- **无缓存**：100 × 50ms = 5,000ms（5秒）
- **80%缓存命中率**：20 × 50ms + 80 × 2ms = 1,160ms（1.16秒）
- **性能提升**：**4.3x**

实际使用中，缓存命中率通常在80-90%。

### 内存占用

- 单条消息Widget：约100KB
- 100条缓存：约10MB
- 可接受的内存换取速度
