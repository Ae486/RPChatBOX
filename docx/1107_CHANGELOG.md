#### 4. 🧪 **模型检测功能**

##### 4.1 检测流程
```
点击"检测"按钮
  ↓
所有模型卡片进入待测试状态
  ├─ 蓝色边框
  ├─ 能力图标旋转动画（360度）
  └─ 显示"👆 touch_app"图标
  ↓
点击某个模型卡片
  ↓
顶部显示提示框："正在测试 GPT-4..." + 加载动画
  ↓
调用 testProvider() API
  ↓
成功/失败消息显示 3 秒后自动消失
```

##### 4.2 动画效果
**方案A（已实现）**：
- **边框闪烁**：`AnimatedContainer` 实现
  - 测试模式：蓝色边框（1px → 2px）
  - 正在测试：蓝色粗边框（2px）
- **图标旋转**：`AnimatedRotation` 实现
  - 旋转角度：`turns: 1`（360度）
  - 动画时长：800ms
  - 曲线：Curves.easeInOut

##### 4.3 测试反馈 UI
**横向提示框**（顶部）：
- **检测中**：🔵 蓝色背景 + 加载动画
  ```
  🔵 正在测试 GPT-4...   [✕]
  ```
- **成功**：🟢 绿色背景 + ✅
  ```
  ✅ 测试成功
  响应时间: 234ms
  可用模型数: 15
  ```
- **失败**：🔴 红色背景 + ❌
  ```
  ❌ 测试失败
  401 Unauthorized: Invalid API Key
  ```

**特性**：
- ✅ 显示详细信息（响应时间、错误详情）
- ✅ 3 秒后自动退出测试模式
- ✅ 可手动关闭
